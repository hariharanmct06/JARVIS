# JARVIS Voice Dashboard
Virtual Chief of Staff and Operations Dashboard for SUJOK AI BOT & HARI BOT & BUSINESS SOLUTIONS.

## Features
*   **Speech-to-Text & Text-to-Speech:** Full voice conversational interface using Web Speech API.
*   **Glowing UI:** Pulsing Arc Reactor visualization changing colors dynamically based on speech states.
*   **Operational Dashboard:** Monitor status reports and pending tasks for Sujok AI and Hari Bot.
*   **Settings Controller:** Customize TTS voice accents, speed, and pitch.

## Getting Started
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the local server:
   ```bash
   npm run dev
   ```
3. Open `http://localhost:5173/` in Google Chrome or Microsoft Edge.
